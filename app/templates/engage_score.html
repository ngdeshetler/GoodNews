Engagement Score

An explanation of the Engagement Score metric

The need for an Engagement score

In this dataset there are five main engagement metrics to look at:

  • Total Views
  • Total Visitors
  • Average Read Time
  • Number of Social Media Shares
  • Number of Comments
Each one of these metrics can provide different insights to how users are engaging with the article. However, it can be difficult to get a broader picture of the user engagement with so many different features and metrics to compare. One total Engagement Score would provide an overview of the general engagement based on each feature.

Data Science and Machine Learning to create an Engagement Score

As you can see in the below chat, which plots each of the five metrics against each other, that most of the metrics are fairly similar to each other. Articles that have more visitors also tend to have more social media share, etc.



This similarity between the five metrics allows us to use a machine learning approach called a Principal component analysis to reduce these metrics down to one main "component". In the initial test case (data from March 2016 - May 2016) this component explained 63% of the total variance in the five metrics.

The Engagement Score

This component is the Engagement Score. Features that are high on the Engagement Score will tend to be high on most of the other five metrics. The score has been scaled so that 0 is the lowest amount of engagement and 10 is the highest, but the Engagement metric does not directly map back on to any one of the other metrics, and so metric is unitless, and the scores are only meaningful in relation to other scores.
